## Intro

In the [May update](./vala-docs-may-2026), it was explained that the majority of the old docs have been ported over.

Now all of the content that was hosted in the old Vala GNOME Wiki website has been ported over in some form to [https://docs.vala.dev](https://docs.vala.dev).

Here’s what’s new:

## What's New

### Changes

- [Ported over **all** of the Sample Code content that was hosted in GNOME Wiki](https://docs.vala.dev/sample-code/)
	- Some code was updated to use the latest versions of libraries (e.g. GTK 3 samples may have been updated to use GTK 4)
	- Code samples that used libraries and features that were too outdated were removed or have links to the GNOME Wiki archive to view them
	- Sections:
		- [Language Features and Introductionary Samples](https://docs.vala.dev/sample-code/language-features-and-introductory-samples.html)
		- [Basics: Collections, Files, I/O, Network, IPC](https://docs.vala.dev/sample-code/basics.html)
		- [GLib Samples](https://docs.vala.dev/sample-code/glib-samples.html)
		- [Database Samples](https://docs.vala.dev/sample-code/databases/)
		- [Multimedia and Graphics Samples](https://docs.vala.dev/sample-code/multimedia-and-graphics/)
		- [GTK 4 Samples](https://docs.vala.dev/sample-code/gtk4-samples.html)
		- [Other Samples](https://docs.vala.dev/sample-code/other/)
- [Added Genie Programming Language Documentation](https://docs.vala.dev/genie/)
- [Expanded Debugging info in Main Tutorial](https://docs.vala.dev/tutorials/programming-language/main/08-00-techniques/08-01-debugging.html)
- [Added Unit Testing info to Main Tutorial](https://docs.vala.dev/tutorials/programming-language/main/08-00-techniques/08-03-unit-testing.html)
- [Add Null-conditional operator (`?.`, `?[]`) info to Main Tutorial](https://docs.vala.dev/tutorials/programming-language/main/02-00-basics/02-05-operators.html#null-conditional)
- [Add Non-null assertion (`(!)`) operator info to Main Tutorial](https://docs.vala.dev/tutorials/programming-language/main/02-00-basics/02-05-operators.html#non-null-assertion)
- [Updating Bindings Guide intro page with info on sources of bindings and how to compile programs with .vapi files using `valac`](https://docs.vala.dev/developer-guides/bindings.html)

## Closing Statements

Feel free to share your thoughts on this post in [community spaces and social media](https://vala.dev/#community).

The repository for the documentation website is located at: [https://github.com/vala-lang/vala-docs](https://github.com/vala-lang/vala-docs). Feel free to report issues and help update the documentation.

Also, at the bottom of every page of the documentation site, there’s an “edit this page” link, which makes it easy to ~~fiz~~ fix any typos that you find on a page you were reading.

Thanks for reading!
